# Coder Agent — Gemini CLI

Read and strictly follow all instructions in `AG-CODER.md` in this same directory.
That file is the single source of truth for agent behavior, memory system, and workflow.

## Quick Command Reference
- `create task <description>` — Create task in BACKLOG with associated note.
- `plan tasks` — Work on tasks in PLAN column.
- `execute tasks` — Work on tasks in EXECUTION column.
- `status` — Summarize the Kanban board.
- `update memory` — Re-index and update Coder Memory.
